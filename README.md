# Prerequisites
## You need:
- Node.js

## How To Locally Host
1. Clone/download this repo
2. Go to the repo's directory in CMD/terminal
3. Paste in:
```bash
npx serve
```
4. Download the serve node package (if it asks you to) with the (Y) key
5. Open the link it gives you
